# ✨ 每日激励 · Daily Spark 桌面版

> 每天在屏幕中央弹出一句激励人心的话，让每一日都充满动力。

Windows 桌面应用，支持自定义语句库、定时弹窗提醒、系统托盘后台运行、导入导出等功能。

## 功能一览

| 功能 | 说明 |
|------|------|
| 🖥️ **屏幕中央弹窗** | 到设定时间自动在屏幕正中弹出激励语，半透明毛玻璃卡片设计 |
| 🔄 **后台运行** | 关闭主窗口后最小化到系统托盘，不影响日常操作 |
| ⏰ **每日定时提醒** | 自定义提醒时间，到点自动弹出 |
| 🚀 **开机自启** | 可选开机自动启动，确保不错过每日提醒 |
| 📚 **语句管理** | 添加、删除自定义激励语句，支持 `\n` 换行符 |
| 📥 **导入 / 导出** | 支持 JSON、TXT、CSV 三种格式，方便备份与迁移 |
| 🎨 **活力 UI** | 深色主题、渐变色彩、动态背景图形、流畅动画 |

## 快速开始

### 安装（推荐）

1. 从 [Releases](https://github.com/your-username/daily-spark/releases) 下载最新版安装包 `每日激励 Setup x.x.x.exe`
2. 双击安装，可选择安装目录
3. 安装完成后桌面和开始菜单会出现「每日激励」快捷方式

### 从源码运行

```bash
# 1. 克隆仓库
git clone https://github.com/your-username/daily-spark.git
cd daily-spark

# 2. 安装依赖（已配置国内镜像，加速下载 Electron）
npm install

# 3. 生成应用图标
npm run icons

# 4. 启动应用
npm start
```

> **注意**：首次运行 `npm install` 时会下载 Electron 二进制文件，耗时可能较长，请耐心等待。

### 2. 设置每日提醒

1. **修改提醒时间**：点击时间输入框，选择你希望弹窗的时间（如 09:00）。
2. **开启/关闭提醒**：点击「启用每日屏幕弹窗提醒」开关。
3. **开机自启**：建议开启「开机自动启动」，确保应用在后台持续运行，到点弹窗。
4. 修改设置后会自动保存，底部状态文字会同步更新。

### 3. 管理语句库

**添加语句：**

1. 在输入框中输入激励语句文本
2. （可选）填写作者名称
3. 点击「➕ 添加」按钮或按 Enter 提交
4. 语句会自动添加到列表顶部

> **提示**：文本中可以使用 `\n` 换行，渲染时会自动折行显示。例如：
> ```
> 第一行文字\n第二行文字
> ```

**删除语句：**

- 每条语句右侧有 🗑️ 删除按钮，点击即可删除

**切换今日语句：**

- 点击今日激励卡片中的 🎲 按钮，可随机切换到另一条语句

### 4. 预览弹窗效果

点击主界面右上角的「🔔 预览弹窗」按钮，立即在屏幕中央看到弹窗效果：

```

弹窗操作：
- 点击「收到，加油！💪」按钮关闭
- 点击右上角 ✕ 按钮关闭
- 按键盘 `Esc` 或 `Enter` 键关闭

### 5. 系统托盘操作

关闭主窗口后，应用不会退出，而是缩小到系统托盘（任务栏右下角）。

| 操作 | 效果 |
|------|------|
| 左键双击托盘图标 | 打开设置主窗口 |
| 右键单击托盘图标 | 弹出菜单：「打开设置」「预览今日弹窗」「退出」 |

托盘图标是一个小闪电 ⚡ 图标，鼠标悬停显示「每日激励」提示。

### 6. 导入 / 导出语句

**导出：**

1. 点击语句库右上角的「📤 导出」按钮
2. 选择保存位置和文件名（默认 `每日激励_YYYY-MM-DD.json`）
3. 保存为 JSON 文件，方便备份

**导入：**

1. 点击语句库右上角的「📥 导入」按钮
2. 选择要导入的文件（支持 JSON、TXT、CSV 格式）
3. 导入的语句会追加到现有语句库的最前面

## 导入格式说明

### JSON（推荐）

完整格式（含可选版本和导出时间）：

```json
{
  "version": 1,
  "exportedAt": "2026-07-03T09:00:00.000Z",
  "quotes": [
    { "text": "第一行\n第二行", "author": "作者名" },
    { "text": "没有作者的语句", "author": "" }
  ]
}
```

也支持纯数组格式：

```json
[
  { "text": "一句话", "author": "作者" },
  { "text": "另一句话" }
]
```

> 字段兼容：`text` / `content` / `quote` 均可作为语句内容；`author` / `source` 均可作为作者。

### TXT（纯文本）

每行一条语句，不支持作者：

```
每一个不曾起舞的日子，都是对生命的辜负。
星光不问赶路人，时光不负有心人。
行动是治愈恐惧的良药。
```

### CSV

`语句,作者` 格式，每行一条：

```
每一个不曾起舞的日子，都是对生命的辜负。,尼采
星光不问赶路人，时光不负有心人。
行动是治愈恐惧的良药。,罗宾·夏玛
```

## 开发与构建

### 开发模式运行

```bash
npm start
```

### 打包为 Windows 安装包

```bash
# 先清理旧的构建文件（会自动结束正在运行的进程）
npm run build
```

打包完成后，安装包位于：

```
release/每日激励 Setup 1.0.0.exe
```

### 只生成目录（不打包）

```bash
npm run build:dir
```

生成的免安装版位于 `release/win-unpacked/`，可直接运行 `每日激励.exe`。

### 生成图标

```bash
npm run icons
```

## 数据存储

用户数据（语句库、设置、提醒记录）存储在：

```
%APPDATA%/daily-spark/daily-spark-data.json
```

例如：`C:\Users\<用户名>\AppData\Roaming\daily-spark\daily-spark-data.json`

> 卸载应用时该文件不会自动删除。如果需要完全清除数据，可手动删除此文件。

## 常见问题

**Q：关闭主窗口后应用还在吗？**
A：在的。关闭窗口只是隐藏到系统托盘，应用仍在后台运行。右键托盘图标选择「退出」才会完全退出。

**Q：如何完全退出应用？**
A：右键系统托盘图标 → 点击「退出」；或者在任务管理器中结束 `每日激励.exe` 进程。

**Q：为什么到了提醒时间没有弹窗？**
A：请检查：① 提醒开关是否开启；② 系统托盘图标是否存在（应用是否在运行）；③ 是否有其他窗口全屏遮挡了弹窗（弹窗是置顶的，但仍可能被全屏游戏/应用覆盖）。

**Q：更换电脑后如何恢复语句库？**
A：在原电脑上点击「导出」保存 JSON 文件，再到新电脑上点击「导入」选择该文件即可。

**Q：支持多显示器吗？**
A：支持的。弹窗默认在主显示器（Primary Display）的屏幕中央弹出。
